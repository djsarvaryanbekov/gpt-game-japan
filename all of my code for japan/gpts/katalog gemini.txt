СИСТЕМАТИЗИРОВАННЫЙ КАТАЛОГ ВОЗМОЖНОСТЕЙ (Версия 2.0)
(Объединяет все наши идеи)
А. МИР И АТМОСФЕРА
A1. Основа Мира: Карта мира, изначально скрытая "Туманом Незнания". Туман рассеивается по мере прохождения ключевых заданий/боссов, открывая новые регионы.
A2. Структура: Мир делится на Крупные Регионы (10-12 шт.), каждый = 15-20 уроков MNN и имеет свой уникальный визуальный стиль и атмосферу.
A3. Локации внутри Регионов:
Города: Центральные хабы. Сложные уроки (Додзё), основные квесты, NPC, Библиотеки/Архивы.
Деревни: Места для практики, ежедневных миссий, мини-игр на повторение, NPC с быстрыми квестами.
Храмы/Башни/Руины: Локации Финальных Испытаний Региона (Боссы).
Арены/Лабиринты: Зоны для самопроверки на время или с ветвлением.
Рынки Знаний / Магазины: Обмен/покупка (за игровую валюту!) подсказок, свитков, косметики.
Сады Мудрости / Оазисы Знаний / Центры Повторения: Места для SRS, рефлексии, просмотра статистики.
Скрытые Локации (Пещеры/Тропы): Открываются по секретным условиям, содержат редкие награды или адаптивные квесты.
Мастерские Перевода: Практика составления фраз.
А4. Навигация: Игрок управляет персонажем на карте (клики/стрелки), перемещаясь по связанным путям между доступными узлами.
Б. ИГРОК И ПРОГРЕССИЯ
Б1. Роль Игрока: "Ученик-Языкопроходец" / "Искатель Знаний" (с возможностью выбора аватара - Ниндзя/Самурай/другой?).
Б2. Основная Прогрессия: Открытие регионов карты после победы над Боссами Регионов.
Б3. Система Опыта: XP начисляется за правильные ответы, прохождение уроков, квестов, испытаний.
Б4. Уровни: Повышение уровня при накоплении XP. Открывает доступ к новым регионам/квестам/возможностям кастомизации. Визуальные/звуковые эффекты левел-апа.
Б5. Достижения / Свитки Знаний: За прохождение уроков, испытаний, выполнение квестов, достижение вех (100 слов, 50 кандзи). Отображаются в профиле. Могут давать бонусы.
В. ЗАДАНИЯ И ОБУЧЕНИЕ
В1. Ядро Уроков (Додзё): 3-панельный интерфейс (AI-Объяснение, Интерактивное Задание, Новые Слова). Использует контент MNN N5-N4, генерируемый AI.
В2. Типы Заданий (в Уроках и Испытаниях): multipleChoice, fillInBlank, перевод RU<>JP (ввод текста), составление фраз (drag-n-drop), соединение пар, (будущее: аудирование?).
В3. Боссы Регионов (Финальные Испытания):
Многоступенчатые: Комбинируют разные типы заданий по всем темам региона.
Динамические/Адаптивные: Могут фокусироваться на слабых местах игрока.
Символические: Битва со "Знанием", "Забвением", "Ошибкой".
Ставки: Возможность рискнуть (потерять XP/прогресс при провале) ради большей награды.
В4. Квесты:
Ежедневные/Еженедельные: Простые задачи на регулярность (от AI-Системы).
Секретные/Адаптивные: Запускаются по условиям (стиль игры, ошибки, время входа). Дают уникальные награды/испытания.
В5. SRS (Система Интервального Повторения): Критически важный инструмент для запоминания слов и кандзи. Интегрирован (в Городах/Садах). AI может напоминать о необходимости повторения.
Г. AI-СИСТЕМА (НАСТАВНИК)
Г1. Воплощение: Не голос/дух, а визуальное представление самой системы/интерфейса. Может проявляться как голограммы, терминалы, светящиеся символы, парящий дрон/сфера (вдохновляясь референсами). Весь UI – это и есть проявление Системы.
Г2. Функции:
Выдача заданий (уроки, квесты).
Объяснение грамматики/лексики (через вызов Gemini).
Проверка ответов.
Отслеживание прогресса (XP, уровни, ачивки, слова/кандзи в SRS).
Предоставление статистики.
Контекстные подсказки (если игрок застрял).
Адаптация (предложение секретных квестов, фокус на слабых темах).
Мотивационные сообщения / поздравления.
Г3. Стиль Взаимодействия: Не навязчивый, не ведущий за руку. Предоставляет информацию и инструменты, но решения принимает игрок. Тон – информативный, поддерживающий, возможно, с нотками тайны или технологичности.
Д. ТЕХНИЧЕСКАЯ БАЗА
Д1. Frontend: HTML, CSS, JavaScript (без фреймворков пока).
Д2. AI: Google Gemini API (ключ ТРЕБУЕТ защиты бэкендом!).
Д3. Хранение: localStorage.
Д4. Backend (План): Serverless (Vercel/Netlify) для прокси ключа – приоритет после базовой навигации.
