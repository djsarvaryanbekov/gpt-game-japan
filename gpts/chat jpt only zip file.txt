**Отчет о проекте "Японский Учитель-Игра"**

## 1. Введение
Проект **"Японский Учитель-Игра"** — это веб-приложение для изучения японского языка, включающее элементы геймификации, систему уровней и достижений, а также интерактивные тесты. Оно разрабатывается для обучения от начального уровня до N4 с использованием учебника *Minna no Nihongo*.

## 2. История разработки

### **Ранние обсуждения и постановка целей**
- **2025-04-02** – Пользователь решил создать кастомного GPT-учителя и веб-приложение для изучения японского языка.
- **2025-04-02** – Определены основные функции: тесты, проверки ошибок, система прогресса, напоминания о занятиях.
- **2025-04-02** – Решено следовать структуре сайта CodeDex.io и сделать темный дизайн.

### **Основные этапы разработки**
1. **2025-04-02** – Разработка концепции и функционала.
2. **2025-04-02** – Определены технологии:
   - **Frontend:** HTML, CSS, JavaScript
   - **AI:** Google Gemini API
   - **Хранение данных:** localStorage
   - **Backend:** Пока не реализован, но запланирован для API безопасности.
3. **2025-04-02** – Реализованы:
   - Запросы к API Gemini для генерации уроков.
   - Обновление интерфейса на основе ответов ИИ.
   - Проверка ответов в тестах и начисление опыта (XP).
   - Система уровней и навигация по урокам.
4. **2025-04-02** – Запланированы дальнейшие улучшения:
   - Улучшение обратной связи на ошибки.
   - Добавление новых типов упражнений.
   - Внедрение системы повторений (SRS), квестов и достижений.
   
## 3. Структура приложения
- **Главный экран** – Кнопка "Старт" для начала обучения.
- **Верхняя панель** – Прогресс, меню, профиль.
- **Словари:**
  1. Обычный словарь – изучение слов.
  2. Канзи-словарь – изучение иероглифов.
- **Геймификация:**
  - Опыт (XP), уровни, достижения.
  - Квесты ("Мастер Канзи", "Грамматический Ниндзя").
  - Тайные задания и бонусы.
- **Технологии:**
  - Frontend: HTML, CSS, JavaScript.
  - API: Google Gemini.
  - Локальное хранилище (localStorage).

## 5. Заключение
Проект находится в стадии разработки, но уже реализованы ключевые функции. В дальнейшем предстоит работа над улучшением обратной связи, расширением типов заданий и добавлением новых игровых элементов.
